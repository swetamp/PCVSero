PCV Serotyping Analysis - Manuscript

Compiled July 1, 2020 by Sweta Patel

Updated January 21, 2021 by Sweta Patel

This markdown file contains the code and output for the results included in the PCV serotyping manuscript. There is a separate R file containing the data cleaning code used to generate the csv files used here. Please note that we ran a number of different models while working on this project; all prior models and code are saved in dated files in the Box folder.

General order of steps:

    1. Load clean csv files and R packages
    1. Get numbers for Table 1 (demographics), Figure 1 (serotype class by year), and supplemental table 1 (serotypes by year)
    1. Statistical analysis of serotype trends over time using multinomial logistic regression

Analysis of serotype trends over time

We will use a multinomial logistic regression model and include age and HIV status as covariates.

Unadjusted analysis

Two serotype classes (PCV-13 vs non-PCV-13), year as a continuous variable. NOTE: per discussion with MK on 7/27, changed reference group to non-PCV13 for ease of interpretation in the manuscript

sero_cov$ca13 <- relevel(sero_cov$ca13, ref = "Not_PCV13")
test1 <- multinom(ca13 ~ diff_y2, data = sero_cov)
## # weights:  3 (2 variable)
## initial  value 185.763444 
## final  value 165.153642 
## converged
summary(test1)

## multinom(formula = ca13 ~ diff_y2, data = sero_cov)
## 
## Coefficients:

## (Intercept)  1.1369463 0.22834257
## diff_y2     -0.4697704 0.07825673

z1 <- summary(test1)$coefficients/summary(test1)$standard.errors
z1
## (Intercept)     diff_y2

#Calculate p-values using Wald tests (here z-tests).

p1
##  (Intercept)      diff_y2 
## 6.387223e-07 1.937775e-09

The exponentiated regression coefficients are relative risk ratios for a unit change in the predictor variable

## extract the coefficients from the model and exponentiate
exp(coef(test1))
## (Intercept)     diff_y2 
##   3.1172348   0.6251458
#Obtaining 95% CI for our RR
exp(confint(test1))
##                 2.5 %    97.5 %
## (Intercept) 1.9925258 4.8768013
## diff_y2     0.5362522 0.7287752

Interpretation:

The relative risk ratio for every 1 year increase since the start of the study for being a PCV13 serotype was 0.63 (95% CI: 0.54, 0.73) > AKA the risk of a PCV13 serotype compared to a non-PCV13 serotype was 37% lower with each year that passed since the start of the study (P<0.0001).
